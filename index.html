Coding4noobies Session1 Task list

  • Create first item on the list
  • Bonus: strike-out the already done tasks
  • Open the index.html file in Chrome
  • Open with live server index.html page and make some changes to see live reload in action
  • Write 2 paragraph at the bottom of the page containing bold, italic and underlined words
  • Bonus: find 3 other tags that allow you to style text and use them in the paragraph
  • Create a new webpage containing an image (local image and internet image)
    • Create a new file image.html
    • Use a template to generate the basic html
  • Exploring JSFiddle
    • Open JSfiddle and copy&paste index.html in the html field
    • Preview the code
    • Share the code snippet with someone else
  • Create a link on this page that opens the image page in a new tab
  • Create a new web page containg a contact form
    • Create firstname input of type text
    • Create email input of type email
    • Create age input of type number
    • Create input of type submit
    • Create gender select dropdown
  • Create a link on this page that opens the form page in a new tab
  • Bonus: make the firstname field mandatory
  • Bonus: add a default value to the email field
  • Bonus: restrict the minimum age to 18 years
  • Bonus: Create a new webpage and embed a youtube video in it
  • Bonus: Create a link on this page that opens the youtube page in a new tab
